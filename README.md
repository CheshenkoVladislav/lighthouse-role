# Ansible Role: lighthouse

Данная роль предназначена для установки и настройки Lighthouse.

## Требования

Роль была протестирована на следующих операционных системах:
- Fedora
- Ubuntu

## Переменные роли

На данный момент роль не использует переменных.

## Зависимости

Нет.

## Пример использования

Ниже приведен пример того, как использовать эту роль в вашем плейбуке:

    - hosts: servers
      roles:
         - lighthouse-role
